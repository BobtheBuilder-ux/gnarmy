DOCUMENT FOLDER

This folder should contain the following PDF files referenced in /data/documents.json:

1. company-profile.pdf
2. export-compliance.pdf
3. product-specifications.pdf

Please add your actual PDF documents here with the exact filenames listed above.

The Document Center page (/document-center) will display these PDFs with:
